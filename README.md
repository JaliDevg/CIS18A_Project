### Running the Code:
1. Open files into a text editor or an IDE.
2. Save the files onto computer.
3. Open 'Command Prompt' (Windows) or 'terminal' (MacOS & Linux) and type into terminal:
```java
javac Main.java
java Main
```
4. The java program will be executed and run.
